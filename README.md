# Gender-Unemployment-Gap-Analysis
Analysis of gender unemployment gap using Power BI

## 📊 Overview
This project analyzes gender disparities in unemployment rates across countries using Power BI.

## 📌 Key Insights
- Female unemployment is higher than male unemployment on average
- Countries like Egypt show the highest gender gap

## 🛠 Tools Used
- Power BI
- Excel

- .pbix file (Power BI report)
